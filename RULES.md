# 프로젝트 수행 규칙 (RULES)

이 파일은 AI 에이전트가 작업을 수행할 때 반드시 따라야 하는 규칙을 정의합니다.

## 1. 배포 및 동기화 (Critical)
- **트리거**: 코드가 변경되는 모든 작업 완료 시.
- **행동**:
  1. `clasp push`를 실행하여 Google Apps Script에 배포.
  2. `git push`를 실행하여 GitHub에 동기화.
  - 순서: `clasp push` -> `git add` -> `git commit` -> `git push`

## 2. 언어 설정
- **커밋 메시지**: 반드시 **한국어**로 작성한다.
- **워크플로우 문서**: 마크다운 내용은 **한국어**로 작성한다.
- **Thinking Process**: 에이전트의 내부 사고 과정(Thinking Mode)은 **한국어**로 작성한다.

## 3. 프롬프트 인식
- 사용자가 프롬프트를 입력할 때마다 이 규칙들을 최우선으로 인식하고 준수한다.
